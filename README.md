# NetSc0ut

## Overview

NetSc0ut is a Raspberry Pi 5 based portable network reconnaissance platform designed for Wi-Fi, Bluetooth, LAN discovery, RF analysis, and hardware expansion.

## Current Features

- Flask web interface
- Touchscreen kiosk mode
- Tool framework
- Wi-Fi scanning
- Bluetooth scanning
- LAN device discovery
- Channel analyzer
- WireGuard VPN integration
- Terminal (ttyd)
- ESP32 companion tools
- nRF24 support
- GPS support
- Modular C++ and Python tools
